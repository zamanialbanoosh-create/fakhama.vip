# Fakhama VIP Rewards System

صفحات:
- مشتری: /
- چک کد زیبا: /check.html?code=TAJ-123456
- اسکن کارمند: /staff.html
- مدیریت جوایز: /admin.html

نصب:
npm install
npm start

متغیرهای مهم:
SHOP_NAME=TAJ AL FAKHAMA
SHOP_WHATSAPP=965XXXXXXXX
ADMIN_PIN=1234
STAFF_PIN=1111

نکته: اطلاعات در پوشه data ذخیره می‌شود. روی هاست رایگان ممکن است بعد از Redeploy ریست شود.
